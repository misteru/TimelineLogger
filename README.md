# 📋 タイムライン日記 / Timeline Logger

> シンプルで使いやすい、ブラウザだけで動くタスク記録ツール。
> A simple, browser-only task logging tool that works everywhere.

---

## ✨ 特徴 / Features

- **ゼロインストール** — HTMLファイルをブラウザで開くだけ。アプリ不要。
- **どのデバイスでも動作** — スマートフォン・タブレット・PC対応。
- **自動タイムスタンプ** — 記録を追加すると日付と時刻が自動で記録される。
- **クリップボードにコピー** — 全記録を `HH:MM，内容` 形式でワンクリックコピー。
- **完全オフライン** — 外部サーバー・通信不要。データはブラウザ内のみ。

## 🚀 使い方 / How to Use

1. 上記のHTMLファイルのいずれかをダウンロードする。
2. ブラウザ（Chrome / Safari / Firefox など）で開く。
3. 以下の3つのボタンで操作する：

| ボタン | 機能 |
|---|---|
| **＋ 記録を追加** | 入力ダイアログを開き、現在日時を自動記録してメモを追加する |
| **⎘ 全記録をコピー** | 全記録を `HH:MM，内容` 形式でクリップボードにコピーする |
| **✕ 記録をクリア** | 全記録を削除して新しい記録の準備をする |

### ⌨️ キーボードショートカット

- `Enter` — 記録を確定して追加する
- `Shift + Enter` — 入力欄内で改行する
- ダイアログ外をクリック — ダイアログを閉じる

---

## 📋 コピー形式 / Copy Format

「全記録をコピー」を押すと、以下の形式でクリップボードに保存されます：

```
09:30，チームミーティングの議事録を作成した
10:15，クライアントへ見積書を送付した
14:00，バグ修正完了、テスト通過
```

そのままチャットツール・メモアプリ・日報などに貼り付けられます。

---

## ⚠️ 注意事項 / Notes

- データはページを**リロードすると消えます**。長期保存が必要な場合は「全記録をコピー」で別のアプリに貼り付けて保存してください。
- インターネット接続不要。完全にオフラインで動作します。
- 外部ライブラリ・フレームワーク不使用。純粋なHTML/CSS/JavaScriptのみ。
